# 🛠️ Skill 模板

此目录包含创建新技能所需的模板文件。

## 📁 模板结构

```
skill-template/
├── README.md.tmpl       # README 模板
├── SKILL.md.tmpl        # SKILL.md 模板
├── config.json.tmpl     # 配置文件模板
├── install.sh.tmpl      # 安装脚本模板
└── .gitignore.tmpl      # .gitignore 模板
```

## 🚀 使用方式

```bash
# 使用 skill-publisher 创建新技能
~/.openclaw/skills/skill-publisher/scripts/create-skill.sh <skill-name>
```
